## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Documentation

To access the API documentation, open your web browser and navigate to:

http://localhost:3000/api/docs


This will bring you to the Swagger interface, where you can explore the available endpoints, their request/response formats, and any authentication requirements.
